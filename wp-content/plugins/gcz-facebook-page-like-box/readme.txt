== Installation ==

1. Install the plugin through the Plugins interface or by uploading the `facebook-page-like-box` directory to your `/wp-content/plugins/` directory.
2. Activate the Facebook Page Like Box plugin.
3. Add the widget wherever you want through the 'Appearance -> Widgets' interface. Make sure to fill out the page identifier!
